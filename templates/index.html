Authentication

Choose session auth for browser access or JWT for headless integrations. Refresh tokens keep integrations online safely.

JWT Workflow

  1. POST /api/token/ with username/password to obtain access & refresh tokens.
  2. Attach the access token to Authorization: Bearer <token> headers.
  3. Renew using POST /api/token/refresh/ before expiry.
